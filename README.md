# Application de gestion des salles

## Étudiant

## Description
Ce projet est une application Python de gestion des salles réalisée dans le cadre de l’examen final de Programmation II.
L’application utilise MySQL pour la persistance des données,
customTkinter pour l’interface graphique,
et une architecture en couches comprenant les modèles,
l’accès aux données, les services et les vues.
